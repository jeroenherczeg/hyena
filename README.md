# An Artificial Intelligence Web Scraper for Laravel 5

## Installation

This package can be installed via Composer:

``` bash
composer require jeroenherczeg/hyena
```

## Usage

```php
$result = Hyena::visit('google.be')->extract(['name', 'colors', 'images']);
```
